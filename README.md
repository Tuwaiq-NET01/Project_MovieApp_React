# Movies App
Movies App is a React challange to create Movie App and to get the data about movies from TheMovieDB APIs.

(صوره)

## Technologies :
- axios
- Redux
- React hooks
- react-i18next
- bootstrap

## Endpoints :
- /
- /popular
- /search

## installation :
1- ` git clone  (repo link) `
2- ` cd (folder name) `
3- ` cd movieapp `
4- ` npm i `
5- ` npm start `
